# Cats vs Dogs Classification using SVM

## Task

Build a Support Vector Machine (SVM) model to classify images of cats and dogs.

## Dataset

Microsoft Cats and Dogs Dataset

## Technologies Used

* Python
* OpenCV
* NumPy
* Scikit-learn
* Matplotlib

## Workflow

1. Load cat and dog images
2. Resize images to 64x64 pixels
3. Flatten images into feature vectors
4. Split data into training and testing sets
5. Train an SVM classifier
6. Evaluate model performance

## Results

* Model: Support Vector Machine (SVM)
* Kernel: Linear
* Accuracy: 52%
